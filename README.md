# ![logo](./logo.png)

This community node is provided wrapper for n8n API. Has implemented all operations!

# Installation

1. Open your n8n instance and go to the menu "**_Settings_**" -> "**_Community nodes_**"
2. Then click on the "**_Add_**" button and type "**_n8n-nodes-n8n-api_**"
3. Tick checkbox "**_I understand the risks of installing unverified code from a public source_**"
4. Click "Install".

# Usage

provide credentials and choose resource and operation. For more informaction go to **your-instance**/api/v1/docs
